1 real bug caught by a real agent

Round-1 agent enabled-3 noticed the sanctioned-address response violated the advertised Problem+JSON contract. Fixed in the same session. Round-2's verify-fix agent confirmed 10/10 assertions now pass. The catch only happened because agents were cross-checking the skill file against the live API — the audit loop the markdown surface enables.
